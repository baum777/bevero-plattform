# Intent Logbook

This folder contains one intent-memory entry per non-trivial work slice.

The MSPR logbook records **what happened in the repo**.  
The Intent Logbook records **which logic, idea, or architectural intention the work followed**.

Do not store secrets, customer data, raw database exports, or `.env*` contents here.

Use the template at:

```txt
docs/agent-team/templates/intent_memory_entry.md
```
