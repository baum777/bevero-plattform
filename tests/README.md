# Tests

Unit- und Integrationstests liegen in `apps/api/src/**/*.test.ts` und werden mit Vitest ausgeführt.

```
npm run test        # watch
npm run test:ci     # einmaliger Run (CI)
```

Dieses Verzeichnis ist reserviert für zukünftige monorepo-weite Testskripte (z. B. E2E, Smoke-Tests über mehrere Apps).
